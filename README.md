# 🎯 Number Guessing Game

## Overview

This project is a simple command-line Number Guessing Game built with Python. The program generates a random number between 1 and 100, and the player must keep guessing until they find the correct number. After each guess, the game provides a hint indicating whether the guess is too high or too low. It also counts the total number of attempts.

## Features

* Generates a random number between 1 and 100
* Accepts user input from the command line
* Provides "Too High" and "Too Low" hints
* Tracks the total number of attempts
* Beginner-friendly Python project

## Technologies Used

* Python 3
* `random` module

## Learning Outcomes

* Working with the `random` module
* Using `while` loops
* Applying `if-elif-else` conditions
* Taking user input with `input()`
* Counting attempts using variables

## How to Run

1. Clone the repository.
2. Navigate to the project folder.
3. Run the program:

   ```bash
   python number_guessing_game.py
   ```
